# Supporting This Project

This project is free and open source.

It exists to help developers build AI-assisted software with better structure,
less repetition, and more predictable results.

If you find it useful and want to support its ongoing maintenance and improvement,
there are a few simple ways to do so.

---

## GitHub Sponsors (Recommended)

You can support this project via GitHub Sponsors:

👉 https://github.com/sponsors/Alvecomm

GitHub Sponsors supports **monthly contributions**, starting as low as **$1/month**.

Even a small recurring contribution helps:
- keep the project maintained
- improve documentation and templates
- develop related open-source tools
- ensure long-term availability

There are no gated features or paywalls.
Support is optional and always appreciated.

---

## Other ways to support

If sponsoring is not an option, you can still help by:

- Starring the repository
- Sharing it with others who may benefit
- Using it in your own projects
- Providing feedback or suggestions
- Opening issues for improvements or clarifications

Open-source projects grow through usage and trust as much as funding.

---

## Transparency

All sponsorships go toward supporting open-source work under the Alvecomm organization,
including templates, tools, and developer resources.

This project will remain open source regardless of sponsorship level.

---

Thank you for using and supporting this project.
